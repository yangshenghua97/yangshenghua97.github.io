git中的文件分类

​ Git将所有文件分为3类：已追踪的、被忽略的以及未追踪的

  • 已追踪的

    已追踪的文件是指已经在版本库中的文件，或者是已暂存到索引中的文件。如果想将新文件newfile添加到已追踪的文件，执行 git add newfile即可
