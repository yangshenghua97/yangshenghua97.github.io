百分号（%）通配符

%表示任何字符出现任意次数。

上面的SQL检索任意以jet为开头的词，%告诉MYSQL接受jet之后的任意字符，不论它有多少字符。通配符可在搜索模式中任意位置使用，并且可以使用多个通配符

下划线（_）通配符

下划线通配符只匹配单个字符而不是多个字符。
